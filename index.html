GitHub Pages 公開手順

はじめに

GitHub Pagesを使用して、静的サイトを簡単に公開する方法を説明します。

手順

  1. リポジトリを作成

    GitHubにログインし、新しいリポジトリを作成します。

  2. ファイルを追加

    リポジトリにHTML、CSS、JavaScriptファイルを追加します。

  3. GitHub Pagesを設定

    リポジトリの設定ページに移動し、GitHub Pagesセクションを見つけて、Branchを選択して保存します。

  4. 公開されたサイトを確認

    設定が保存されると、リポジトリのURLでサイトが公開されます。
